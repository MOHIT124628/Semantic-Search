## Overview

This project demonstrates a simple semantic search system using Sentence Transformers.
Instead of keyword matching, it understands the meaning (context) of a query and retrieves the most relevant sentences.

## Feacture

1.Convert sentences into vector embeddings\n
2.Perform semantic similarity search
3.Retrieve top matching results
4.Uses cosine similarity for comparison
5.Lightweight and easy to understand implementation

## Tech Stack

1.Python
2.SentenceTransformers
3.Numpy
